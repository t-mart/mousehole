# Mousehole Community Contributions

This directory contains work that integrates Mousehole in a wider ecosystem.
These are **community-contributed and supported on a best-effort basis**.
They're not part of the Mousehole app or its Docker image, may lag behind
releases, and are provided as-is. [`t-mart`](https://github.com/t-mart) may not
be able to support them.

## What's here

- [Homepage Integration](./homepage/)
  - Display network info, API status, and timing data
  - Custom widgets and API integration

- [Unraid Installation Guide](./unraid/)
  - XML template for manual installation
  - README file with installation instructions

## Adding your own

Open a PR adding a subdirectory with a README: what it does and how to use it.

The biggest risk for inclusion is having a maintainer who can support it, so
please do not submit contributions you are not willing to maintain. Stale
contributions may be removed.
