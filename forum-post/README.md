# Forum Post HTML

It is difficult to maintain the Mousehole forum post on the MAM forum.

Therefore, we maintain it here. To import it:

1. Copy the contents of [`./index.html`](./index.html) to the clipboard
2. Click the "Edit" button on the forum post on MAM
3. Under the "View" menu, select "Source code"
4. Replace all content with the clipboard contents
5. Preview and submit.
